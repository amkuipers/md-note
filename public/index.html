md-note

●

    Double-click the canvas (or ＋ Note) to add a note.

    Drag a note's top bar to move it. Drop it onto another note to stack.

    Shift-drag pulls a note out of its stack; click the count badge to unstack all.

    Drop a note onto a desktop in the sidebar to move it there — dragging a stack moves the whole stack. Layouts scale with the window, so they keep their shape on any screen size.

    Click a note's body to edit markdown; click elsewhere to render.

    Click the desktop name to rename it; use the theme selector to relayout.

    With the Node server, notes are saved as .md files in the data/ folder. Without it (static hosting, GitHub Pages, or opening index.html directly), they are saved in this browser's localStorage.

    Export downloads all desktops as one JSON file; Import adds them back — use it to back up or move notes between browsers.
